
## Ejercicios de Introducción a la Programación Web

Este repositorio contiene dos proyectos prácticos desarrollados para dominar las bases de la programacion en front del curso de becalos.

## Contenido del Repositorio

El repositorio se divide en dos secciones principales:

### 1. Programa primero de experimentacion 
Un espacio de experimentación donde se ponen en práctica etiquetas fundamentales y se explica la semantica a usar.
* **Temas cubiertos:** * Uso de encabezados (`<h1>`) y párrafos.
    * Inserción de imágenes y enlaces externos
    * Listas ordenadas (`<ol>`) con enlaces
    * Diferencia entre elementos de bloque (`<div>`) y elementos de línea (`<span>`).
    * Atajos incluidos en comentarios:**
    * `Alt + Shift + A`: Comentar bloque
    * `Alt + Shift + F`: Organizar código 
    * `Alt + Z`: Ajuste de palabra

### 2. The Planets (`planets.html`)
Una página con una estructura más avanzada y semántica que simula un sitio informativo sobre el sistema solar como pratica dentro de la clase con todos los compañeros.
* **Estructura Semántica:**
    * `<header>` y `<nav>` para el menú de navegación
    * `<main>` para el contenido principal 
    * `<footer>` para mostrar estadísticas de los planetas

---

## Tecnologías Utilizadas
* HTML5:** Estructura y semántica.

##  Cómo visualizar los proyectos

1. Clona este repositorio o descarga los archivos.
2. Asegúrate de tener las imágenes en las rutas correspondientes:
   * `./imagen_de_mario.jpg`
   * `./src/tierra_imagen.jpg`
3. Abre cualquiera de los archivos `.html` en tu navegador favorito.
